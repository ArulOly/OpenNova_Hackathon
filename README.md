# 🎓 MindMate – AI-Powered Student Wellness & Productivity Assistant

## 🚀 Overview
MindMate is an innovative solution designed to help students manage stress, improve productivity, and maintain mental well-being using **AI, automation, and IoT**.  
It combines **Python (backend + AI models)**, **JavaScript/HTML (frontend UI)**, and **IoT integration** to create a holistic student support system.

---

## ✨ Features
- 📊 **Smart Stress Detection** – AI analyzes wearable data + journaling sentiment.
- 🧑‍🏫 **Personalized Productivity Coach** – Custom study schedules & reminders.
- 🎮 **Gamified Well-being** – Rewards, streaks, and leaderboards for motivation.
- 🛡️ **Safe Mental Health Support** – Coping strategies + counselor connections.
- 🔒 **Privacy & Security** – Encrypted data storage with student control.
